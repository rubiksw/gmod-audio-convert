# gmod-audio-convert
Encodes a folder of audio files to a format and sample rate that's supported by Garry's Mod and other Source engine games.

Supports the following formats & sample rates:
- .mp3
- .wav
- 44100 Hz
- 22050 Hz
- 11025 Hz
